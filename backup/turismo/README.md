# Introdução

Este é o site da condutora local de turismo Ana Karla Cruz.
Karla atualmente reside em Sobradinho de Minas, distrito de São Tomé das Letras, MG. 
Karla é condutora de turismo local credenciada, desde de 2022, pela prefeitura de São Tomé das Letras. Karla oferece diversos roteiros com translado para os principais pontos turisticos de São Tomé das Letras e Sobradinho.
